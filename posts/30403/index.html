【Vue3 开源项目】后台管理系统（Vue3 + TypeScript + Pinia）

基于 Vue3 全家桶开发的后台管理系统，线上地址在这。适用于自学 Vue3 的同学，以及外包项目快速交付。

技术栈

该项目使用技术栈如下：

  • Vue 3：前端框架
  • TypeScript：语言
  • Vite：打包工具
  • Pinia：状态管理
  • Vue Router：路由管理
  • Element Plus：UI 组件库
  • Echarts：图表
  • Axios：网络请求
